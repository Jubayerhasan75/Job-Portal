# Jobify - Job Portal Website 

**Jobify** is a simple yet professional-looking **frontend-only** job portal website developed using **HTML, CSS, and JavaScript**. It allows users to browse job listings, view job details, and apply by filling out a form. This project does not include any backend or database functionality.

##  Short Description
Jobify is a clean, responsive frontend job portal UI where users can explore job opportunities, read full descriptions, and submit applications via a form. No backend or server-side functionality is used.

##  Why Jobify?
- ✅ Simple but professional design
- ✅ Clean, user-friendly interface
- ✅ Functional form validation (frontend only)
- ✅ Fully responsive on all screen sizes

##  Folder Structure

├── index.html
├── jobs.html
├── job-details.html
├── apply.html
├── contact.html
├── style.css
├── script.js

##  How to Use
1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Navigate through the available pages (Jobs, Apply, Contact).
4. All functionalities are frontend-based (no backend required).
